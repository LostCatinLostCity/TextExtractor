# Text and Document Summarizer 
This project is a text extractor made using easyocr.

# Technologies Used
* Python 3.8 or above
* Streamlit
* PyPDF2
* PyTorch
* EasyOcr
* Pillow
* OpenCV
* Numpy

# Steps to run
 * Clone repository
 * Install pre-requisities and dependencies

 * > streamlit run Ocr1.py
